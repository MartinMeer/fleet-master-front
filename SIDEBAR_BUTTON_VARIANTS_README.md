# 🎨 Sidebar Button Design Variants

Три красивых варианта дизайна для четырех кнопок в боковой панели вашего приложения "My Car Tech Tracker".

## 📋 Описание вариантов

### 1. 🔮 Стеклянный эффект (Variant 1)
- **Стиль:** Современный glassmorphism
- **Особенности:** 
  - Полупрозрачный фон с размытием
  - Тонкие границы и тени
  - Эффект "блика" при наведении
  - Плавные анимации масштабирования

### 2. 🎯 Неоморфизм (Variant 2)
- **Стиль:** Мягкий, тактильный дизайн
- **Особенности:**
  - Мягкие тени для создания глубины
  - Эффект "нажатия" при клике
  - Градиентные фоны для каждого цвета
  - Тактильные ощущения

### 3. 🌈 Градиентные карточки (Variant 3)
- **Стиль:** Цветные градиенты с иконками
- **Особенности:**
  - Яркие градиентные фоны при наведении
  - Тонкие иконки в углу каждой кнопки
  - Современная типографика
  - Эффект "всплытия" карточек

## 🚀 Как использовать

### Автоматическое применение
Варианты автоматически загружаются при запуске приложения. Ваш выбор сохраняется в localStorage.

### Программное переключение
```javascript
import { SidebarButtonVariants } from './src/sidebarButtonVariants.js';

// Применить конкретный вариант
SidebarButtonVariants.applyVariant('variant-1'); // Стеклянный эффект
SidebarButtonVariants.applyVariant('variant-2'); // Неоморфизм
SidebarButtonVariants.applyVariant('variant-3'); // Градиентные карточки
SidebarButtonVariants.applyVariant('default');   // По умолчанию

// Переключить на следующий вариант
SidebarButtonVariants.cycleVariant();

// Получить текущий вариант
const current = SidebarButtonVariants.getCurrentVariant();
```

### Доступные константы
```javascript
SidebarButtonVariants.VARIANTS.DEFAULT        // 'default'
SidebarButtonVariants.VARIANTS.GLASSMORPHISM  // 'variant-1'
SidebarButtonVariants.VARIANTS.NEUMORPHIC     // 'variant-2'
SidebarButtonVariants.VARIANTS.GRADIENT_CARDS // 'variant-3'
```

## 🔧 Технические детали

### Файлы
- `src/styles/sidebar-button-variants.css` - CSS стили для всех вариантов
- `src/sidebarButtonVariants.js` - JavaScript утилита для управления
- `public/sidebar-variants-demo.html` - Демо-страница

### CSS Переменные
Каждый вариант использует CSS переменные для цветов:
```css
--gradient-bg: linear-gradient(...);
--button-icon: "🔧";
```

### Производительность
- Оптимизированные CSS-анимации
- Минимальное влияние на производительность
- Плавные переходы с `cubic-bezier`

## 🎨 Настройка цветов

Для изменения цветов отредактируйте CSS переменные в `sidebar-button-variants.css`:

```css
.sidebar-buttons.variant-1 .sidebar-button-green {
  background: linear-gradient(135deg, rgba(34, 197, 94, 0.1) 0%, rgba(22, 163, 74, 0.05) 100%);
  border-color: rgba(34, 197, 94, 0.3);
}
```

## 🔄 Миграция

Если у вас уже есть кастомные стили для кнопок, они будут автоматически переопределены при применении варианта. Для совместимости используйте более специфичные селекторы.

## 📝 Примечания

- Все варианты совместимы с существующим функционалом
- Состояния кнопок (hover, active, focus) сохранены
- Доступность (accessibility) не нарушена
- Поддержка клавиатурной навигации
